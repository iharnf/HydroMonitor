# История изменений

## 0.1.3

- Исправлены пины TFT для M5StickC Plus2: DC=GPIO14, RST=GPIO12 (были ошибочно GPIO23 и GPIO18 от оригинального M5StickC PLUS).
- Убран display_backlight — на Plus2 нет AXP192, подсветка всегда включена.

## 0.1.2

- Возвращен дисплей через компонент `st7789v`.
- Используется модель `TTGO TDisplay 135x240`, близкая к экрану M5StickC Plus2.
- Сеть, API, web server и чтение GPIO36 оставлены без изменений.

## 0.1.1

- Временно отключен дисплей для диагностики запуска.
- Оставлены Wi-Fi, API, web server и чтение GPIO36.
- Цель версии: проверить, что устройство стабильно отвечает по сети.

## 0.1.0

- Добавлена первая прошивка ESPHome для M5StickC Plus2.
- Добавлено чтение аналогового выхода PH-4502C на GPIO36.
- Добавлен вывод отфильтрованного напряжения электрода в mV на дисплей.
- Включены Wi-Fi, API, OTA, web server и диагностика Home Assistant.
- Добавлена закомментированная заготовка под DS18B20.
